# Serial Toolkit

Serial Toolkit is a cross-platform desktop app for serial port debugging and testing.

## Features

- List and select available serial ports
- Configure baud rate, data bits, stop bits, parity, flow control
- Connect/disconnect with status feedback
- Send data (text or HEX) and view received data (text or HEX)
- Timestamped log output with a large, virtualized log window
- Toggle RTS/DTR and enable periodic (timer) sending
- One-click copy logs
